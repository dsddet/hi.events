## Tại sao chọn BrothersBit?

Hầu hết các nền tảng bán vé tính phí cho mỗi vé và khóa dữ liệu của bạn vào hệ sinh thái của họ. **BrothersBit là một giải pháp thay thế mã nguồn mở hiện đại cho Eventbrite, Tickettailor, Dice.fm, và các nền tảng bán vé khác** dành cho những người tổ chức muốn kiểm soát hoàn toàn thương hiệu, thanh toán, dữ liệu và cơ sở hạ tầng.

Được xây dựng cho người quảng bá giải trí đêm, người tổ chức lễ hội, địa điểm sự kiện, nhóm cộng đồng và người tổ chức hội nghị.

### Docker

```bash
git clone git@github.com:dsddet/Hi.Events.git
cd Hi.Events/docker/all-in-one

# Tạo khóa (Linux/macOS)
echo "APP_KEY=base64:$(openssl rand -base64 32)" >> .env
echo "JWT_SECRET=$(openssl rand -base64 32)" >> .env

docker compose up -d
```

> [!TIP]
> **Người dùng Windows:** Xem `./docker/all-in-one/README.md` để biết hướng dẫn tạo khóa.

Mở `http://localhost:8123` và tạo tài khoản của bạn.
