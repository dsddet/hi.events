## Dlaczego BrothersBit?

Większość platform do sprzedaży biletów pobiera opłaty za bilet i zamyka Twoje dane w swoim ekosystemie. **BrothersBit to nowoczesna, otwarta alternatywa dla Eventbrite, Tickettailor, Dice.fm i innych platform** dla organizatorów, którzy chcą pełną kontrolę nad marką, procesem checkout, danymi i infrastrukturą.

Stworzony dla promotorów życia nocnego, organizatorów festiwali, sal koncertowych, grup społecznych i gospodarzy konferencji.

### Docker

```bash
git clone git@github.com:dsddet/Hi.Events.git
cd Hi.Events/docker/all-in-one

# Wygeneruj klucze (Linux/macOS)
echo "APP_KEY=base64:$(openssl rand -base64 32)" >> .env
echo "JWT_SECRET=$(openssl rand -base64 32)" >> .env

docker compose up -d
```

> [!TIP]
> **Użytkownicy Windows:** Zobacz `./docker/all-in-one/README.md` aby uzyskać instrukcje generowania kluczy.

Otwórz `http://localhost:8123` i utwórz swoje konto.
